title: WebDevice.GetSharedFile Method

#### Syntax
<pre class="syntax"><code class="avr"> **BegFunc GetSharedFile Access(*Public) Type([WebDisplayFile](web-display-file-class.html))

#### Parameters
<dl>
        <dt>
 *shareFileName*

</dl>
